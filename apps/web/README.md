# finance-jars Web

Frontend React + Vite de la app local-first.

## Entorno

Copiar variables ejemplo:

```bash
cp .env.example .env
```

Variable principal:

- `VITE_API_URL` (default esperado: `http://localhost:3000`)

## Comandos

```bash
npm install
npm run dev
```

## Calidad

```bash
npm run lint
npm run build
```

## Rutas principales

- `/dashboard`
- `/frascos`
- `/ingresos`
- `/gastos`
- `/movimientos`
- `/configuracion`
