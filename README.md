# finance-jars
Una app **local-first** para organizar ingresos, gastos y frascos de dinero de forma simple, rápida y privada.  ✨ **Stack:** React · NestJS · Prisma · SQLite   🎯 **Objetivo:** uso personal, cero costo, diseño minimalista   🔐 **Auth local** · 💸 **Gastos por frasco** · 📊 **Dashboard mensual** · 🌙 **Dark mode**
